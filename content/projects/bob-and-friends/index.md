PAI (Personal AI Infrastructure) is an ADHD-optimized AI system built on top of Claude Code. What started as a fork of Daniel Miessler's PAI project has evolved into a fully custom skill system, memory architecture, and multi-agent workflow engine running across every project in the network.

Wally runs as Bob Prime. Specialized agents — Howard (design), Bill (infrastructure), Mario (engineering), Hugh (philosophy) — each have their own project context, personality, and domain expertise. The Algorithm (v3.7.0) governs how complex work gets broken down, tracked, and verified.
